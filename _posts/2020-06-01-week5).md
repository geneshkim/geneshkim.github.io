Week 5: Without much notice at all, I suddenly found myself with two extra roommates this week. It was disappointing to not have the entire four room apartment to myself, but I was glad to have a place to stay in Boston and have an in-person experience. Faraz and I set up some programs to help with segmentation of STL files to allow for decomposed descriptions of individual components. Unfortunately I had some accessibility challenges with VS Code's terminal to remote ssh into the MIT machines. I heard that another blind person (a researcher himself) has found some success with VS Code, so I set up a meeting with them to learn their work arounds. However, at this point, it was becoming increasingly apparent to me that we wouldn't be able to develop the system we had in mind using existing data sets.
I started evaluating whether or not it would be worth obtaining a data set using Amazon Mechanical Turk. But 
I put a pause on these thoughts late in the week to start preparing for AccessHack, an accessibility-based hack-a-thon hosted by Northeastern and Hamilton College. It was the first hack-a-thon I've attended, so I wasn't sure exactly what to expect.
