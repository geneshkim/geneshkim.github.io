I returned from convention and began implementing part of the 3D model interpretation pipeline. I learned about the structure of STL mesh files as compared to point cloud representations of 3D models. I built a program to extract angled images from an STL so that we would have data to feed into a computer vision model that would describe images. I reconnected with Faraz, who informed me that the tools we had tested didn't go as well. They seemed to be writing descriptions that felt more animated, like "a picture of an alien posing", which isn't precise or modular enough for 3D modeling. We decided to explore other models and data sets.
